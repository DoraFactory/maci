# MACI 文档

MACI (Minimal Anti-Collusion Infrastructure) 是一个基于密码学的链上投票系统，通过零知识证明提供隐私保护和抗串谋能力。

## 核心组件

### 智能合约

- **Registry** - 管理 Operator 注册和投票轮次创建
- **AMACI** - 匿名投票合约实例，处理用户注册和投票

### SDK 和工具

- **@dorafactory/maci-sdk** - TypeScript 客户端 SDK
- **@dorafactory/maci-operator** - Operator 系统
- **maci-crypto** - Rust 密码学库

### 零知识电路

使用 Circom 实现的 ZK 电路，支持 Groth16 和 PLONK 证明系统。

## 适用场景

- **DAO 治理** - 社区提案和协议参数投票
- **二次方融资** - Gitcoin Grants 类型的公共物品资助
- **组织选举** - 内部选举和代表选举
- **代币投票** - 基于持币量的加权投票
